# Speed-Typing-Game
A speed typing game using HTML, CSS, and Javascript. This game will help you increase your typing speed. You can set your target WPM in the input field, and the game will set countdowns based on that input and the amount of text you need to type. 

SETUP:
1. Clone this repository.
2. Open it in your favourite code editor.
3. Open it with a live server.
4. Start Typing!
